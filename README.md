## About Project
In this repository, there is a notebook in which I have trained a model to predict crop according to given data like 
* Sodium
* Potassium
* Phosphorus
* Temperature
* Humidity
* pH
* Rainfall

It will predict from following types of crops.

* Chickpea
* Mungbean
